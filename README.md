## 🎮 Features (EN)

### 🗺️ Polygon Drawing Mode
Fly through the game with the camera, left-click to add points, and press Enter to save. Press Backspace to undo the last point.

### 🖥️ Modern NUI Menu
A sleek, blue-themed management panel on the left side. List, edit, and delete zones.

### ✏️ Editing Support
Reopen saved safezones and update their points whenever you want.

### 📏 Vertical Safezone
Not just on the ground; it provides extended protection upwards and downwards (adjustable via config).

### 🔐 Permission System
* Access via Rockstar license list  
* Or admin / god permissions (QB, Qbox, ESX)  

### 🌍 8 Languages Supported
Turkish, English, German, French, Portuguese, Polish, Spanish, Dutch

### 📢 Discord Webhook
Zone creation, update, and deletion actions are logged to Discord (player name, Discord ID, coordinates, etc.).

### ⚙️ Easy Config
All settings are in a single file: `config.lua`

### 🔒 Escrow Protected
The source code is encrypted; only the config file remains open.
